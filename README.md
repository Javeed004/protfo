
# Javeed Zulfikar — Personal Portfolio Website

A clean, responsive personal portfolio website built with vanilla HTML, CSS, and JavaScript — showcasing projects, skills, and contact information for a Data Enthusiast based in Tamil Nadu, India.

## 🚀 Features

- 🧑‍💻 **Hero Section** — Catchy introduction with a call-to-action button
- 💼 **Projects Showcase** — Alternating layout highlighting key AI & ML projects
- 🛠️ **Skills Section** — Technical and soft skills listed clearly
- 📖 **About Me** — Personal bio with a link to the resume
- 📬 **Contact Section** — One-click email CTA for quick outreach
- 📱 **Responsive Design** — Mobile-friendly layout across all screen sizes
- 🔗 **Social Links** — GitHub and LinkedIn in the footer

## 📦 Getting Started

No build tools or package managers required. Just clone and open!

### 1. Clone the Repository

```bash
git clone https://github.com/Javeed004/protfo.git
cd protfo
```

### 2. Open in Browser

Simply open `index.html` directly in your browser:

```bash
# macOS
open index.html

# Linux
xdg-open index.html

# Windows
start index.html
```

Or use the **Live Server** extension in VS Code for hot-reload during development.

## 📝 Customization

To personalize this portfolio for yourself:

1. Update your name and bio in `index.html`
2. Replace project details, links, and screenshots in the `work` section
3. Edit the skills lists under the `skills` section
4. Swap out images in the `images/` folder
5. Replace the resume PDF in `resume/`
6. Update social media links in the footer
